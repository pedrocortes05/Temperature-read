# Setup
```shell
pip install -r requirements.txt
```

# Use
```shell
python main.py
```

Options
--folder
    Select folder location for videos

--show-plot
    Show plots

--save-plot
    Save plots

--save-frames
    Save image every 5 seconds

```shell
python main.py --folder videos

python main.py --save-plot

python main.py --show-plot

python main.py --save-frames
```
